# Dataset Information

Dataset Source:
Kaggle Crop Yield Dataset

Purpose:
Used for agricultural productivity forecasting.

Input Features:

- Crop Type
- Location
- Temperature
- Rainfall
- Soil Information
- Previous Yield Data


Output:

Predicted Crop Yield (kg/hectare)


Data Processing:

Python Pandas will be used for:

- Cleaning missing values
- Feature selection
- Preparing data for Machine Learning model
