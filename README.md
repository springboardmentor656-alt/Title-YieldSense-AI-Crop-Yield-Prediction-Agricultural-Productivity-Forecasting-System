# YieldSense AI

## Crop Yield Prediction & Agricultural Productivity Forecasting System

YieldSense AI is an AI-powered platform designed to predict crop yield and improve agricultural productivity using machine learning, weather data, soil parameters, and historical crop information.

## Features

- User Registration
- User Login
- JWT Authentication
- PostgreSQL Database Integration
- FastAPI Backend
- Crop Yield Prediction
- Agricultural Insights and Recommendations

## Tech Stack

### Backend
- FastAPI
- Python

### Database
- PostgreSQL
- SQLAlchemy

### Authentication
- JWT
- Passlib (bcrypt)

### Machine Learning
- Scikit-Learn
- Pandas
- NumPy
