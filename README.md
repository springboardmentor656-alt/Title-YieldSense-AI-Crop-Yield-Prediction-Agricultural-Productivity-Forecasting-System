# YieldSense AI

YieldSense AI is a predictive analytics platform designed to forecast agricultural yields and assess climate risks. By leveraging geographical data, historical weather patterns, and soil composition metrics, the system provides actionable intelligence before resources are committed to the field.

This repository encompasses a modernized **FastAPI** backend, a sleek **Next.js** frontend with a responsive glassmorphism UI, and a robust **PostgreSQL** database architecture.
