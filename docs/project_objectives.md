# YieldSense AI

## Project Objective
To develop an AI-powered platform that predicts crop yield using historical agricultural data, weather conditions, soil information, and machine learning techniques.

## Features
- User Registration and Login
- Crop Yield Prediction
- Weather Analysis
- Soil Analysis
- Dashboard and Reports
- AI-Based Recommendations

## Workflow
1. User logs in.
2. User enters farm details.
3. System processes weather and soil data.
4. Machine learning model predicts crop yield.
5. Dashboard displays insights and recommendations.
